# Packaging Line Performance Analysis for a Craft Brewery

## Objective
Analyse the canning line performance for a fictional craft brewery in order to identify where output is being lost, which SKUs are creating the most operational pressure, and how packaging issues are affecting inventory risk and wholesale order fulfilment.

## Business context
Andrae Craft Brewing Co. operates a canning line packaging four core SKUs:
- Core Lager.
- Session IPA.
- Pale Ale.
- Alcohol-Free IPA.

Over the last 12 weeks, the brewery has experienced:
- Inconsistent packaged output.
- Frequent downtime.
- Elevated packaging waste on selected SKUs.
- Periodic late wholesale orders.

Management wants a clearer view of line performance and practical recommendations to improve output and service level.

## Key questions
1. Which SKUs contribute most to packaged output?
2. Where is downtime concentrated?
3. Which SKUs have the highest waste rates?
4. Which products create the highest inventory risk?
5. How often are wholesale orders shipped late or short?
6. What operational changes should management prioritise?

## Scope
This project focuses on:
- Packaging and canning runs.
- Downtime events.
- Finished goods inventory.
- Wholesale order fulfilment.

It does not cover brewing, fermentation, procurement, or finance.

## Tools
- Excel: data audit, checks, and exploratory analysis.
- SQL: cleaning, transformation, joins, and KPI views.
- Power BI: dashboarding and communication.

## Deliverables
- Excel analysis workbook.
- SQL schema and KPI queries.
- Power BI dashboard.
- Case study with findings and business recommendations.
