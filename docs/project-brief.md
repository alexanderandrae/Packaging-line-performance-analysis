# Packaging Line Performance Analysis for a Craft Brewery

## Objective
Analyse canning line performance at Andrae Craft Brewing Co. in order to identify where packaged output is being lost, which SKUs create the most operational pressure, and how packaging performance affects finished goods availability and wholesale order fulfilment.

## Business context
Andrae Craft Brewing Co. is a fictional London craft brewery operating a single canning line across four core packaged SKUs:
- Core Lager.
- Session IPA.
- Pale Ale.
- Alcohol Free IPA.

Over the last 12 weeks, packaging performance has been inconsistent. Output has varied from week to week, downtime has been higher than target, and some SKUs have generated more waste and service risk than others.

Management believes that a combination of downtime, changeovers, and SKU mix may be reducing line efficiency and contributing to late or incomplete wholesale orders. They want a clearer view of where losses are occurring and which actions would most improve output and service level.

## Key questions
1. Which SKUs contribute most to packaged output?
2. Which downtime causes account for the greatest lost time?
3. Which SKUs have the highest waste rates?
4. Which products are most likely to fall below safety stock?
5. How often are wholesale orders shipped late or not in full?
6. What operational changes should management prioritise first?

## Scope
This project focuses on:
- Packaging and canning runs.
- Downtime events.
- Finished goods inventory.
- Wholesale order fulfilment.

It does not cover brewing, fermentation, procurement, or finance.

## Tools
- **Excel**: data audit, checks, and exploratory analysis.
- **SQL**: cleaning, transformation, joins, and KPI views.
- **Power BI**: dashboarding and communication.

## Deliverables
- Excel analysis workbook.
- SQL schema and KPI queries.
- Power BI dashboard.
- Case study with findings and business recommendations

## Success criteria
The project should:
- Identify the main sources of lost output on the canning line.
- Show which SKUs create the most operational and service pressure.
- Quantify downtime, waste, inventory risk, and service performance.
- Provide clear, prioritised recommendations for management.
